# CoffeShopCart

Simple shop cart app created with React, Redux and Ant Design. 

## General Info

I created this app to practise React and Redux. 
You can add random coffe to cart then fill form and order it. 
Have fun :)

### Technologies

Project is created with:
* React 16.13.0
* Redux 4.0.5
* Ant Design 4.0.3
